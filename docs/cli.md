SatoshiTop
=====

> Bitcoin full node monitoring dashboard for terminal

## SYNOPSIS

`satop`


## DESCRIPTION

SatoshiTop is a system monitoring dashboard for Bitcoin full node in terminal much like htop, top, gtop etc...
